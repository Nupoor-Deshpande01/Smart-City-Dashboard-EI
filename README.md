# Nexus Smart City Analysis Dashboard
A cutting-edge, interactive web dashboard designed to visualize the **Futuristic Smart City Citizen Activity Dataset**. It offers a sleek, cyberpunk-inspired UI to analyze urban mobility, energy consumption, and socio-digital rhythms of citizens in a modern smart city ecosystem.

## 🚀 Key Features

*   **Immersive Hero Section:** An interactive, parallax-animated HTML `<canvas>` cityscape background featuring moving neon-rimmed buildings and glowing stars.
*   **Grid View Analytics:** A beautiful glassmorphism-style dashboard overview displaying 6 key data metrics seamlessly integrated with Chart.js.
*   **Dynamic CSS Heatmap:** A custom-built, interactive HTML/CSS heatmap visualizing the daily lifestyle breakdown (Work, Sleep, Leisure, Shopping) across the 7 days of the week.
*   **Inline Focus Mode:** Global toggle system (`Grid View` | `Focus Mode`) allows users to dive deep into any specific chart with dedicated descriptions and "Critical Insights" inline, completely replacing disruptive pop-ups. Navigational arrows allow cycling through metrics.
*   **Zero-Dependency Architecture:** Built completely from scratch using vanilla HTML5, CSS3, and JavaScript within a single lightweight file (`nexus_city.html`). No build pipelines required.

## 📊 Dataset Insights Explored
*   **Mobility:** Mode of Transport distribution (EVs vs Gas vs Public Transit vs Walking).
*   **Sustainability:** Home Energy Consumption per sector.
*   **Health:** Correlation map tracking Steps Walked against Calories Burned.
*   **Daily Routine:** Matrix heatmap of time allocation per day throughout the entire week.
*   **Environmental Impact:** Historical timeline of Carbon Footprint (kg CO2) trends.
*   **Social Behavior:** Stacked bar breakdown comparing digital isolation (Social Media) vs Public physical events.

## 🛠️ Tech Stack
*   **Core:** HTML5 & Vanilla CSS3 
*   **Styling:** Custom fluid Grid, Flexbox, UI Animations, Glassmorphism, and Parallax backgrounds.
*   **Interactivity:** Vanilla JavaScript (Canvas rendering, DOM manipulation, Layout state toggling)
*   **Data Visualization:** Chart.js 4.4.1 (CDNs)
*   **Fonts:** Google Fonts APIs (`Orbitron`, `Outfit`)

## 💻 How to Run
This is a completely standalone application! 

You don't need any complex setup, python backend, node modules, or build steps.
1. Download or locate `nexus_city.html`.
2. Double-click the file to open it in any modern web browser (**Chrome, Firefox, Safari, Edge**).
3. Scroll down perfectly immersive data visualizations!
